<task type="auto">
  <name>Task 1: Create picker module and wire into init.lua</name>
  <files>lua/session/picker.lua, lua/session/init.lua</files>
  <action>
Create `lua/session/picker.lua` following the `local M = {}` / `return M` module pattern used by all session modules.

The module exposes one function: `M.show_picker(window, pane)` which:

1. Loads manager via `require("lua.session.manager")` and gets current workspace via `wezterm.mux.get_active_workspace()`.

2. Calls `manager.list_sessions()` to get sessions sorted by most recent.

3. Builds a `choices` array for InputSelector:
   - For each session: `{ id = session.name, label = prefix .. session.name }` where prefix is `"* "` for current session, `"  "` (two spaces) for others.
   - If no sessions exist, add a single informational choice: `{ id = "__empty__", label = "No sessions found" }`.
   - ALWAYS append a sentinel create entry as the last choice: `{ id = "__create__", label = "+ Create new session..." }`.

4. Calls `window:perform_action(wezterm.action.InputSelector({...}), pane)` with:
   - `title = "Sessions"` (per user decision)
   - `choices = choices` (built above)
   - `fuzzy = true` (enable fuzzy matching)
   - `action = wezterm.action_callback(function(inner_window, inner_pane, id, label) ... end)` (nested callback)

5. The nested selection callback handles:
   - `id == nil`: User pressed Escape, return (no-op).
   - `id == "__empty__"`: No sessions, return (no-op).
   - `id == "__create__"`: Show `PromptInputLine` asking for session name. In the PromptInputLine callback, if the user enters a non-empty name, call `manager.create_session(name)`. On failure, show toast. On success, log info.
   - `id == current` (where current = active workspace name captured in outer closure): Silent no-op, return.
   - Otherwise: Call `manager.attach_session(id)`. On failure (`ok` is nil), show toast via `inner_window:toast_notification("Session Error", err or "Failed to attach", nil, 3000)`.

6. For the PromptInputLine in create-if-not-found:
   ```lua
   inner_window:perform_action(
     act.PromptInputLine({
       description = "New session name",
       action = wezterm.action_callback(function(win, p, line)
         if not line or line == "" then return end
         local ok, err = manager.create_session(line)
         if not ok then
           win:toast_notification("Session Error", err or "Failed to create session", nil, 3000)
         end
       end),
     }),
     inner_pane
   )
   ```

Then update `lua/session/init.lua` to expose the picker module:
- Add `local picker = require("lua.session.picker")` after the existing manager require.
- Add `M.picker = picker` after `M.manager = manager`.
